FAQ

Where do my requests go?

Requests appear in your Netlify Forms dashboard (your admin inbox). You can also enable email notifications.

Can you accept payments here?

Yes, but don’t start there. First, get your pipeline organized. Next step: Stripe checkout links or invoices, and a policy page to reduce disputes.

How do we reduce PayPal chargebacks?

Use clear terms, verification steps, and ideally migrate to card/Stripe with identity checks. Also keep all communication inside one channel (email tickets).

Can I add “in-stock” listings later?

Yes. We can add an inventory page fed by a simple JSON file or a headless CMS, and let customers request a specific listing.
